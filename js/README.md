## 概要
- SQSコンシューマのサンプル *js/ts*
    - 受信メッセージ/ペイロードをコンソール出力する.

## 前提
- `Bun` （ランタイム）で確認済（`Bun` を使用してます）.
- LocasStack起動していないと動きません.
- `re-queue` キューへのメッセージを受け取ります.

## セットアップ
1. ライブラリインストール
    ```
    bun i
    ```
1. `.env` 配置（コピー）
    ```
    cp .env.sample .env
    ```

## 動作確認手順
1. LocalStack起動
1. 実行
    ```
    bun run consumer.ts
    ```

## 他
- `Ctrl-c` で停止.
